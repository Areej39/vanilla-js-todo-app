# Vanilla JS Todo App

A simple task manager built with HTML, CSS, and vanilla JavaScript. Add, delete, and filter tasks.

## Features
- Add/delete tasks
- Mark tasks complete with strikethrough  
- Filter: All, Active, Completed
- Responsive layout
